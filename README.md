# Молоток Бот - Telegram бот для строительного магазина

Telegram бот для магазина строительных материалов "Молоток" в Воронеже.

## Функциональность

- Приветственное меню с кнопками
- Информация об ассортименте товаров
- Контакты магазина  
- Акции и скидки
- Прием заказов
- Ответы на частые вопросы

## Технологии

- Python 3.8+
- python-telegram-bot
- Развертывание на Railway

## Контакты магазина

- Адрес: г. Воронеж
- Телефон: +7 (958) 509-44-99
- Часы работы: Ежедневно 8:00-19:00
